# Zehn (ذہن)

Zehn is an AI-powered emotional reflection engine—a digital sanctuary designed to translate your internal states into aesthetic artifacts. It acts as a "Wiser Cousin," offering empathetic, culturally grounded (specifically Pakistani/Lahore contexts), minimalist, and insightful reflections without the clinical tropes of traditional journaling apps.

## 核心 Features (Core Engines)

- **The Sada (سدا) Analysis:** Multimodal audio processing that detects tone (Melancholy, Hype, Zen, Chaos) and energy levels—understanding not just what you say, but *how* you feel. It adapts the UI spacing, speed, and colors dynamically.
- **The Qalam (قلم):** A poetic synthesis engine that converts your vibe into a traditional Urdu Couplet (Sher) spanning both Roman Urdu and native script, packaged with a soul-heavy English translation.
- **The Aks Engine (عکس):** Visual generation that synthesizes the emotional subtext of your entry into a high-fidelity, ethereal, or "Y2K Streetwear" aesthetic art prompt.
- **Nuskha (نسخہ):** The recalibration agent that provides expansive, actionable micro-interventions (Cognitive Reframing, Somatic Grounding, Sensory Shifts) deeply rooted in cultural context (e.g., *pehli baarish*, late-night stillness). 
- **Halaat Dashboard (حالات):** A dynamic interface that maps your cumulative emotional footprint to color theory (Electric Blue for High Energy/Stress, Mustard Yellow for Peace, etc.).

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS, Framer Motion (for purposeful, fluid animations)
- **Icons:** Lucide React
- **AI/Intelligence:** Powered by Gemini via `@google/genai`
- **Audio/Voice:** Web Audio API & MediaRecorder for raw, unstructured vocal reflections.

## Development Setup

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Philosophy

Zehn is built on the philosophy of **"Vibe Coding"**—moving away from rigid syntax and technical overhead to focus purely on the essence, feeling, and flow of creation. The interface avoids "AI-isms" ("I understand how you feel") in favor of concise, impactful, and vibe-coded language.
